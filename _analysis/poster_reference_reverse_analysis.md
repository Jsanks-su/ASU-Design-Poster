## Quantitative Signals
- Most references are vertical poster formats; wide formats are exceptions, not the default.
- Bottom-edge complexity is frequently high, consistent with credit blocks, title blocks, billing text, or dense lower composition.
- Several strong posters use very high blank/low-detail fields with one concentrated high-contrast mechanism; others use dense graphic systems, but both usually preserve a clear title hierarchy.
- The reference set is not consistently minimalist: it includes bold commercial color, large titles, graphic symbols, collage, and highly legible poster typography.
- Therefore a prompt that says no readable text by default or over-optimizes for quiet negative space will drift away from the reference set.

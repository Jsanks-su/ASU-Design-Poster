# Typography System

Typography must follow the style route. Do not use one default title style for every subject.

Read `typography-composition-catalog.md` when the title risks becoming only horizontal or vertical. Choose a composition role for the type before choosing the font style.

## Default For Direct Image Generation

For ChatGPT Image 2 or direct poster-generation tests, the poster should visibly contain typography by default:

- main title treatment
- context-appropriate supporting copy: tagline, quote, theme phrase, annotation, column label, metadata, side note, footer text, release-like microtype, bilingual fragments, or small designed labels
- lower or side information block, invented emblem, logo-like mark, icon, pictogram, seal, or tiny metadata cluster when it fits the brief

Exact small text does not need to be readable. It should read visually as intentional poster copy. Small movie-poster-style text can be used for beauty, rhythm, atmosphere, and hierarchy. Use full fake cast lists, false real release dates, or billing blocks only when the user explicitly asks for full release simulation. Otherwise use generic or fictive microtype, field labels, annotation, footer marks, or small metadata texture. Keep invented marks generic/fictive unless the user supplies real logos or assets.

## Microcopy And Marks

Use microcopy as design material, not only as literal information:

- tiny English or Chinese lines that read as production notes, chapter labels, coordinates, case metadata, field notes, dates, warnings, or poetic fragments
- invented studio-like emblems, logo-like marks, icons, pictograms, certification seals, map ticks, waveform marks, stamps, or tiny diagrams
- footer strips, side columns, marginalia, top-corner labels, circular locks, barcode-like patterns, or small caption grids

Choose one microcopy system that matches the poster world. A space poster may use coordinates, signal data, and a tiny probe icon. A wildlife poster may use field-note marks and habitat coordinates. A coming-of-age poster may use soft handwritten fragments. A noir poster may use case labels or neon-sign marks. Avoid real brand/studio logos unless supplied by the user.

## Decision Rules

| Poster route | Recommended typography |
| --- | --- |
| Hard sci-fi, AI, technology, space, cyber-thriller | English main title plus Chinese secondary title; condensed modern sans-serif, wide tracking, metallic white/silver, strict grid alignment. |
| Prestige contemporary drama | Quiet serif or refined grotesk; small subtitle; restrained title block with generous negative space. |
| Documentary / social realism | Editorial typography; plain sans-serif or sober serif; credible, small, and precise. |
| Chinese historical, wuxia, folk, ink-style drama | Chinese main title can use calligraphy, seal-script, stele, handwritten brush, ink wash, or rubbing texture; avoid generic computer calligraphy. |
| Hong Kong noir/action/retro | Bold Chinese display type, hand-painted one-sheet lettering, neon sign letterforms, condensed English export subtitle. |
| Japanese minimalist drama/horror | Small, quiet typography; vertical or asymmetric title placement; restrained Japanese-poster spacing logic. |
| Anime theatrical / animation | Drawn title lettering, soft rounded sans, hand-lettered subtitle, or dramatic theatrical logo treatment; match the world and age group. |
| Cartoon / family animation | Friendly rounded or hand-drawn lettering, bright but controlled color, clear hierarchy; avoid clipart/children's worksheet feel. |
| Modernist / graphic / Saul Bass-like | Typography can become structure: huge letters, cut shapes, rotated blocks, strict grid, limited palette. |
| Polish poster / surreal / expressionist | Hand-rendered, rough, distorted, or painterly lettering can carry mood; keep it intentional and readable. |
| Ink / line drawing / white space | Small seal-like marks, vertical Chinese, brush title, or spare serif/sans; let blank space breathe. |
| Horror | Unease through spacing, erosion, scratch, silence, or asymmetry; avoid dripping blood cliche unless parody. |
| Romance / youth | Softer serif, handwritten, rounded, or light sans; emotional but not influencer-template. |
| Comedy / satire | Deadpan grotesk, awkward spacing, playful hand lettering, or ironic formal type; avoid meme-font shortcuts. |

## Title Hierarchy Patterns

### Bilingual Technology / International

```text
Primary title: exact English title, large, clean, cinematic, usually uppercase.
Secondary title: exact Chinese title, smaller, aligned with the English block.
Typography: condensed modern sans-serif, wide tracking, strict grid.
```

### Chinese-First Cultural

```text
Primary title: exact Chinese title.
Typography: calligraphy, seal-script, stele inscription, ink brush, weathered print, or hand lettering chosen by period and mood.
English subtitle: small, optional, used as international/export title or secondary hierarchy.
```

### Graphic / Abstract

```text
Primary title: exact title can become image structure.
Typography: oversized, cropped, rotated, stacked, sliced, or repeated with discipline.
Image and type should feel designed together, not layered afterward.
```

### Animation / Cartoon

```text
Primary title: expressive title logo or hand-lettered title.
Typography: rounded, painted, cel-style, comic-style, or storybook depending on the age and tone.
Keep readability and avoid low-end mascot branding.
```

### Post-Production Typography

Use this only when the model struggles with exact Chinese text and the user prioritizes typography accuracy over a complete one-step poster:

```text
No exact readable title rendered. Reserve a clean title-safe area with poster hierarchy; keep an annotation, side-note, or footer-copy zone only if the image still needs visible poster structure.
```

## Anti-Low Typography

Avoid:

- generic bold poster fonts with no relationship to genre
- fake 3D bevel, chrome, gold embossing, and glowing outlines
- huge slogan text that makes the poster feel like a social media thumbnail
- random font mixing
- random icons, fake logo clutter, or microcopy that fights the main title
- unreadable AI-generated Chinese as the only title
- overusing calligraphy because the title is Chinese
- title text centered by default when the route needs asymmetry, movement, or silence
- fully text-free output when the goal is a finished poster test

Repair:

- choose one title hierarchy before prompting
- choose one title composition role: image, slash, seal, marginalia, footer, environmental signage, whisper, repeated grid, or post-production safe area
- specify exact text only when the model can render it well
- include visible title treatment and context-appropriate supporting copy for direct image-generation tests
- add small text, marks, icons, or logo-like elements when they improve poster presence and remain subordinate to the main hierarchy
- reserve a clean title-safe area only when post-production typography will be better
- make English carry the main typographic burden when the style is international sci-fi or technology
- let hand-lettering, brush, cartoon, or abstract type appear only when it matches the selected route

# Cinematic Poster Quality Bar

Use this before finalizing every serious poster direction, especially when the output feels merely usable, predictable, or like a dressed-up social media image. The goal is not more detail. The goal is a sharper visual mechanism.

When a mechanism does not emerge quickly, read `poster-invention-operators.md` first. When several mechanisms seem plausible, judge them with `aesthetic-judgment-rubric.md` before writing the final prompt.

The reference standard is not "quiet minimalism". A strong poster may be restrained, loud, commercial, illustrated, typographic, character-led, or graphic. Judge whether it has poster presence, not whether it is tasteful in the safest way.

## One-Image Mechanism

Every strong poster needs one primary image mechanism. Pick one before writing the prompt:

| Mechanism | What it does | Prompt cue |
| --- | --- | --- |
| Negative-space reveal | The absence forms the idea. | eye gap, white field, missing body, empty bed, dark doorway, cutout silhouette |
| Scale pressure | Tiny human against a vast structure. | small figure under wall, gate, wave, office facade, mountain, smoke column, title block |
| Single symbolic object | One object carries the story. | apple, gun, rose, tennis ball, mask, file, shoe, tooth, train, light, chair |
| Body fragment | A cropped body part becomes the drama. | eyes, hand, mouth, back, neck, wound, feet, hair, silhouette |
| Color-field rupture | A large color plane is broken by one event. | red cut across black, yellow field with black figure, white room split by shadow |
| Typographic image | Type is not a label; it becomes structure. | title as wall, title as road, title as smoke, title as grid, title as scar |
| Material metaphor | Surface carries meaning. | rubbed stone, newspaper, hospital curtain, velvet, skin, ash, film grain, map, ledger |
| Optical trap | The viewer must resolve the image. | reflection, inversion, double exposure, split profile, impossible shadow, spiral, hidden figure |
| Editorial crop | A severe crop creates tension. | close eyes, half face, cut-off object, subject pushed to edge, massive blank area |
| Pattern with anomaly | Repetition is broken by one meaningful difference. | rows, blinds, windows, trees, seats, files, lines, one rupture |

If the poster cannot name its mechanism, it is probably only an illustration.

## Obvious Poster Test

Before accepting the first idea, name the obvious poster in one sentence. The obvious poster is usually:

- the main character looking dramatic over an atmospheric background
- the theme symbol placed in the center with a title underneath
- a literal scene from the story
- a collage of important plot elements
- a fashionable style applied to the topic

Use the obvious poster only when it already has unusual compression, tension, and specificity. Otherwise, apply an invention operator: substitution, scale shift, negative space, material transfer, typographic mutation, viewer-position shift, pattern anomaly, or boundary violation.

## Tension Tests

Before final output, identify at least one tension pair:

- huge vs tiny
- blank vs crowded
- beautiful vs violent
- calm surface vs hidden threat
- flat graphic shape vs human consequence
- old material vs modern subject
- bright color vs dark meaning
- precise grid vs emotional disorder
- body intimacy vs social scale
- readable commercial hook vs art-film restraint

Name the tension in the design rationale. This prevents attractive but weightless posters.

## Poster Presence

A finished poster direction should usually include:

- visible main title treatment
- secondary copy such as tagline, quote, theme phrase, annotation, column label, metadata, microtype, or short supporting line when useful
- footer or side information block, invented emblem, logo-like mark, icon row, pictogram, seal, or tiny metadata cluster when it fits the brief
- one dominant focal hierarchy
- enough visual amplitude to read at thumbnail size

Do not confuse "premium" with low-energy restraint. If the poster feels like a clean concept frame, add poster structure: title scale, supporting copy, annotation/footer text block, release-like microtype, invented marks, stronger crop, color field, graphic symbol, character focus, or pictorial-poster hierarchy.

## Difference Beyond Style

Do not make variants by only changing medium. Vary one of these deeper levers:

- **Image mechanism**: object metaphor vs scale pressure vs negative space
- **Composition law**: central icon, diagonal force, vertical ritual, edge crop, grid, field rupture
- **Viewer distance**: intimate face, observed room, monumental landscape, symbolic object
- **Color logic**: congruent, counterpoint, single accent, monochrome, material color
- **Typography role**: quiet label, title-as-image, marginalia, seal, export subtitle, institutional type
- **Cultural material**: archive, stage, court, street, body, architecture, print, screen

## Upgrade Moves

| Weak result | Upgrade move |
| --- | --- |
| Subject centered over atmospheric background | Replace the background with a mechanism: shadow, field, object, cut, grid, or void. |
| Too many plot elements | Keep one object, one face, one room, or one landscape pressure. |
| Pretty but forgettable | Add contradiction: beauty against damage, order against panic, blankness against violence. |
| Looks like AI key art | Reduce detail density; add real print logic, crop discipline, and a designed title zone. |
| Looks like a campaign graphic | Remove slogans and random callout icons; keep only symbols or microcopy that make the image imply consequence. |
| Looks like a film still | Add poster abstraction: color field, impossible scale, symbolic object, or typographic structure. |
| Looks like fan art | Reduce likeness emphasis; build the poster around a graphic idea. |
| Looks premium but samey | Change the mechanism, not the texture. |

## Minimum Bar

Before finalizing, answer yes to all:

- Can the poster be described in one sentence without listing many objects?
- Is there a named image mechanism?
- Did you compare it against the obvious poster?
- Is there a clear tension pair?
- Is the composition doing more than placing a subject in the center?
- Is typography assigned a role before the prompt asks for text?
- Does it include visible poster typography by default for direct generation?
- Would the poster still have a clear image idea if the title were removed?
- Does the color choice have a narrative job?
- Does it have enough poster presence and visual amplitude?
- Is there a reason this subject needs this style, not just a fashionable look?

## Prompt Add-On

Append this structure to poster prompts when quality needs to be pushed:

```text
Image mechanism: [negative-space reveal / scale pressure / symbolic object / body fragment / color-field rupture / typographic image / material metaphor / optical trap / editorial crop / pattern with anomaly].
Visual tension: [one tension pair].
Composition law: [central icon / diagonal force / vertical ritual / edge crop / strict grid / vast field / layered screen / spiral / split profile].
Poster copy: visible main title treatment plus context-appropriate supporting text; use microtype, invented emblems, icons, seals, footer or side information blocks when they improve hierarchy and fit the brief.
The poster should read as a finished high-quality poster or pictorial poster, not a scene illustration, social media graphic, or text-free concept plate.
```

# Anti-Low-Design Gate

Use this before finalizing every poster prompt. Do not confuse non-realistic with low quality. Cartoon, anime, ink, abstraction, collage, and painterly posters can be premium when the style is coherent.

## Hard Reject

Reject or remove these unless the user explicitly asks for a deliberately tacky parody:

- generic "AI poster" look: glossy plastic faces, random fantasy detail, oversharpened skin, fake depth
- cheap social-media template: huge slogan block, stickers, badges, arrows, explosive callouts
- ecommerce style: product collage, price-tag energy, shiny reflection floor, random floating icons
- uncontrolled cyberpunk: purple/blue neon everywhere, random code rain, glowing city without narrative reason
- uncontrolled color styling: teal-orange by default, random gradients, all-dark prestige sameness, or accent colors with no story role
- fake luxury: gold-on-black templates, bevelled 3D serif title, stock smoke, meaningless crown/diamond symbols
- low-effort horror: red eyes, dripping blood text, skulls, random fog, jump-scare face in the center
- chaotic montage: too many characters/objects, no hierarchy, no negative space
- AI text failures: illegible title blocks, broken Chinese characters, dense fake credit blocks that become visual noise
- oversaturated gradients: rainbow backgrounds, plastic blue-magenta glow, bokeh blobs
- stock-photo obviousness: generic handshake, smiling office people, laptop hero, flat corporate icons
- incoherent style mixing: too many art styles, unrelated medium cues, prompt soup
- accidental sameness: defaulting every subject to dark realistic festival-poster mood
- stale Chinese style: hanfu figure pasted on misty mountains, decorative calligraphy plus red seal, generic lantern/bamboo/cloud/dragon/plum-blossom symbols, gray ink wash filter, antique texture without dramatic mechanism
- generic "premium poster" safety: centered subject, atmospheric background, tasteful title, but no negative-space reveal, scale pressure, symbolic object, color rupture, typographic structure, or other image mechanism
- over-restrained concept plate: no visible title, no supporting copy structure, no poster hierarchy, too much empty tastefulness, or a clean frame that does not feel like a finished poster

Allowed when designed: small release-like microtype, invented emblems, logo-like marks, icons, pictograms, seals, footer metadata, and tiny bilingual fragments. They become low-design only when they are random stickers, fake brand claims, fake cast-list clutter, or unrelated decoration.

## Repair Rules

| Low Signal | Better Movie-Poster Move |
| --- | --- |
| Too many ideas | Pick one symbolic object or one emotional face. |
| Too much text | Reserve a typography zone; add text in post-production. |
| Looks like an ad | Add narrative tension, shadow, silence, and consequence. |
| Looks like a thumbnail | Increase negative space and reduce slogan size. |
| Looks like AI art | Add real lens behavior, film grain, imperfect texture, grounded lighting. |
| Looks like a game cover | Reduce power poses, weapons, sparks, and fantasy UI elements. |
| Looks like a generic tech banner | Replace abstract neon with a physical metaphor and human stakes. |
| Looks too decorative | Make every visual element carry story, conflict, or mood. |
| Outputs look similar without a content reason | Change stance, symbol, composition, typography, color logic, or visual family only where the content supports it. |
| Color feels arbitrary | Assign the palette a job: evidence, wound, hope, corruption, memory, signal, ritual, or danger. |
| Non-realistic route looks childish | Add film-poster composition, mature palette, typography hierarchy, and print texture. |
| Style feels like prompt soup | Keep one medium and one genre influence only. |
| Chinese style looks old-fashioned | Choose a modern Chinese cinematic mechanism: red-black cut, shadow theatre, material archive, palace/gate pressure, contemporary guofeng, or minimal ink force. |
| Poster feels usable but flat | Name one image mechanism and one tension pair before adding detail. |
| Poster feels too quiet | Add poster presence: visible title, supporting copy, annotation/footer text block, stronger crop, larger symbol, richer color field, or character-led visual force. |
| Poster feels under-designed | Add a controlled microcopy system: tiny release-like text, invented emblem, icon row, seal, footer metadata, or side labels that match the poster world. |

## Minimum Quality Bar

Before final output, answer yes to all:

- Is there one dominant poster idea?
- Is there a named image mechanism, not just a mood or background?
- Is there a visual tension pair?
- Is the style route specific enough to guide medium, composition, and typography?
- Does the platform crop have a clear safe area?
- Is the palette intentional, whether restrained or deliberately vivid?
- Does the color strategy support the poster stance: congruent, counterpoint, bold accent, popular-readable, material, painterly, or monochrome?
- Is text handled intentionally?
- Does this include visible poster copy when the goal is direct generation?
- Would the image idea still be strong if the copy were removed?

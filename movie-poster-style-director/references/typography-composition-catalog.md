# Typography Composition Catalog

Use this with `typography-system.md`. Typography is a composition system, not a final label.

## Layout Moves

| Move | Best for | Direction |
| --- | --- | --- |
| Oversized cropped title | graphic, revenge, satire, iconic subjects | Let letters run beyond the canvas edge; image and type share weight. |
| Diagonal slash title | revenge, chase, rupture, action | Align title or bilingual subtitle with a cut, road, river, sword stroke, crack, or beam of light. |
| Title-as-object | ecology, tech, historical evidence, abstraction | Build the title from forest, archive, city blocks, data, smoke, water, fabric, or wounds. |
| Seal / circular lockup | history, ritual, institution, case file | Use stamps, rings, official marks, evidence labels, or coin-like seals. |
| Marginalia / annotation | legal, documentary, investigation, archive | Put small text as notes, case numbers, side comments, red circles, underlines, or evidence tags. |
| Split-column title | bilingual, historical/modern tension | One script anchors one side; the other script becomes subtitle, evidence, or export title. |
| Footer information strip | poster authenticity, editorial hierarchy | Use a narrow footer with metadata, issue labels, barcode-like marks, short notes, or other brief information when it fits the brief. |
| Microcopy constellation | sci-fi, documentary, investigation, archive, ecology | Scatter tiny but aligned text, ticks, coordinates, captions, or data labels as controlled texture around the main image. |
| Logo-like emblem | animation, genre cinema, festival-like posters, institutional stories | Add an invented emblem, seal, pictogram, badge, or small studio-like mark as part of the poster system; do not use real brand logos unless supplied. |
| Icon / pictogram row | adventure, sci-fi, travel, youth, public-world posters | Use 2-5 tiny symbols with captions as rhythm or world-building, not as stickers or callout icons. |
| Whisper title | grief, memory, art-house, horror restraint | Small type in large negative space; silence becomes the event. |
| Repeated title grid | obsession, bureaucracy, time loop, propaganda | Repeat with controlled spacing; degrade, erase, or interrupt one instance. |
| Interrupted title | mystery, censorship, trauma, crime | Cover, tear, burn, crop, blur, redact, or misregister part of the text. |
| Reflected title | dream, water, memory, dual identity | Put text in reflection, glass, shadow, or duplicate layer. |
| Stair / path title | journey, court, mountain, pursuit | Let title follow steps, road, riverbank, skyline, or architectural rhythm. |
| Neon/sign title | Hong Kong noir, city, nightlife | Treat type as environmental signage, not floating overlay. |
| Handwritten trace | intimacy, documentary, confession | Use notes, diary marks, chalk, brush, pencil, or forensic handwriting. |

## Bilingual Patterns

- Chinese main title as image, English as restrained export subtitle.
- English main title for international/tech/noir routes, Chinese as secondary block.
- Chinese vertical title plus English horizontal footer to create period/global contrast.
- English small caps used as case metadata, date, location, or evidence code.
- One script carries emotion, the other carries order.
- Use bilingual friction intentionally: modern English grid against ancient Chinese archive, or Chinese brush against international crime-thriller typography.
- Use microcopy bilingually when helpful: Chinese title for emotion, English data/footer marks for structure; or English title with Chinese side notes for intimacy.

## Selection Signals

- Revenge or violence: diagonal slash, cropped type, red accent, interrupted title.
- Legal or investigative: marginalia, evidence labels, case-number hierarchy, seal marks.
- Chinese historical: vertical brush, stele/rubbing texture, seals, archive columns, but avoid default calligraphy if another typographic argument is stronger.
- Ecology: title-as-landscape, missing letters, root/branch strokes, small renewal accent.
- Technology: strict grid, English main title, small Chinese subtitle, data-like spacing.
- Memory or grief: whisper title, reflection, faded repeated title, negative space.
- Comedy or satire: deadpan grid, awkwardly formal type, bureaucratic lockup, one absurd misalignment.
- Broad commercial appeal: large readable title, strong contrast, footer strip, clear hierarchy.

## Prompt Requirements

Specify:

- title role: dominant image, secondary label, environmental signage, evidence mark, or post-production safe area
- placement: top, side, diagonal, footer, margin, circle, path, reflection, crop edge
- script hierarchy: Chinese-first, English-first, bilingual equal, or metadata-only
- material: brush, ink, stamp, sans-serif, serif, hand-painted, typewriter, stencil, neon, ledger, rubbing
- microcopy system: footer strip, side notes, invented emblem, logo-like mark, icon row, coordinate grid, certification seal, stamp, tiny diagram, or none

When exact Chinese rendering is risky, reserve a clean typography zone and describe the intended layout for post-production.

## Anti-Low Typography

Avoid:

- only centered horizontal title by default
- only vertical calligraphy because the title is Chinese
- random font mixing without hierarchy
- random decorative logos, icons, badges, and microcopy clusters that feel like stickers
- fake 3D bevel, glow, gold, chrome, or AI-broken Chinese
- title pasted on top of a finished image

Repair by making typography participate in the poster idea.

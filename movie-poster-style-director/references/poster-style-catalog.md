# Poster Style Catalog

Use this as a design-language reference, not as a copying instruction. Never ask an image model to reproduce a specific poster, logo, actor, still frame, trademark, or protected character. Extract composition, restraint, tone, and visual grammar.

## How To Use

Choose 1 primary lineage and optionally 1 secondary tension. Translate the user's arbitrary content into that lineage's grammar.

Do not default to one release-era template, but allow release-like microtype when it strengthens the poster. Small legal-line texture, invented emblems, festival-like marks, generic studio-style symbols, tiny date/location/data labels, and footer metadata can improve authenticity and hierarchy. Avoid automatic full cast-credit blocks, false real release claims, and real brand/studio logos unless the user asks for release simulation or supplies those assets. For general prompt work, extract poster composition, title/copy hierarchy, visual tension, material, color, and audience readability.

Example:

- Bad: "Make it like the Parasite poster."
- Good: "Prestige social-thriller poster language: restrained domestic setting, uneasy negative space, symmetrical blocking, muted palette, a single symbolic object, quiet class tension."
